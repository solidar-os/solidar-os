# Existing Software: Rede de Reciprocidade (Portugal)

This document maps the digital tools used by Portuguese reciprocity networks and CSA/AMAP groups to manage collective purchasing and trust-based distribution.

## 1. Specialized Software (Local & Open Source)
These tools were developed within the Portuguese ecosystem to address the specific needs of self-managed groups (Núcleos).

| Name | Status | Technical Notes | Link |
| :--- | :--- | :--- | :--- |
| **Michaca** | **Active** | The primary open-source tool in Portugal. Designed to manage "Reciprocidade" groups, consumer lists, and producer connections with a focus on simplicity. | [michaca.org](https://michaca.org) |
| **Reciprocidade.pt** | Active | A web portal and directory that acts as a connector between producers and "Núcleos," though it is more of a directory than a management suite. | [reciprocidade.pt](https://reciprocidade.pt) |

## 2. Multi-Model Platforms
Platforms used by larger Portuguese collectives or hubs that require more robust logistical features than those provided by specialized local tools.

| Name | Status | Technical Notes | Link |
| :--- | :--- | :--- | :--- |
| **Open Food Network (PT)** | **Active** | Managed by the *Rede para a Soberania Alimentar*. It is the bridge for groups that act as professional hubs while maintaining the reciprocity spirit. | [openfoodnetwork.pt](https://openfoodnetwork.pt) |
| **Cagette.net** | Limited | Used by some groups in the border regions (Alentejo/Spain) or those inspired by the French AMAP model. | [cagette.net](https://cagette.net) |

## 3. Collaborative & Organization Tools
* **Mailing Lists / Riseup:** Many groups still rely heavily on mailing lists for the "moral" coordination of the network.
* **Spreadsheets (Nextcloud/LibreOffice):** Used for the weekly "Cestas" management where Michaca is not yet implemented.
* **Praça:** A commercial but locally-focused marketplace that some individual producers use to reach customers outside the reciprocity network.
