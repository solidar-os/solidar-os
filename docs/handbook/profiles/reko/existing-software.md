# Existing Software: REKO (Nordic European Countries & UK)

This document maps the digital tools used by REKO rings to manage flash-sales and parking-lot distributions, moving away from total dependence on Facebook Groups.

## 1. Specialist Software
These tools are designed to handle the "pre-order and quick pickup" flow without the overhead of a traditional e-commerce.

| Name | Status | Technical Notes | Link |
| :--- | :--- | :--- | :--- |
| **REKO.market** | **Active** | Specifically built for the Nordic market. Allows producers to list products and customers to reserve them for specific ring dates. | [reko.market](https://reko.market) |
| **Local Food Nodes** | Active | (Sweden) An open-source, map-based platform. It allows producers to create "nodes" (pickup points) where consumers can book products. | [localfoodnodes.org](https://localfoodnodes.org) |

## 2. Multi-Model Platforms
Platforms that have adapted their workflow to support the "Flash Market" style of REKO rings.

| Name | Status | Technical Notes | Link |
| :--- | :--- | :--- | :--- |
| **Open Food Network (NO/UK)** | **Active** | Widely used in Norway and the UK to manage REKO-style cycles. It provides a more robust inventory than Facebook comments. | [openfoodnetwork.org.uk](https://openfoodnetwork.org.uk) |
| **NeighborFood** | Active | (UK/IE) Its "Click & Collect" model is often used as a professionalized version of a REKO ring. | [neighborfood.ie](https://neighborfood.ie) |

## 3. Social Media & Communication (The "Legacy" Stack)
* **Facebook Groups:** Still the primary "software" for 90% of REKO rings. Orders are placed as comments on producer posts.
* **Messenger / WhatsApp:** Used for rapid coordination between producers during the 30-60 minute distribution window.
* **Google Forms:** Occasionally used by organizers to register new producers or collect feedback.
