# Existing Software: GDC (Spain / LatAm)

This document maps the digital tools used by Spanish-speaking consumption groups to coordinate orders, logistics, and internal economy.

## 1. Specialist Software
These tools were born within the "Economía Social y Solidaria" (ESS) in Spain to solve the specific needs of local "nodos".

| Name | Status | Technical Notes | Link |
| :--- | :--- | :--- | :--- |
| **Karacol** | **Active** | Open Source (Python/Django). The most iconic tool for Spanish groups. Focuses on self-management and collective logistics. | [karacol.org](https://karacol.org) |
| **Katuma** | Active | Open Source (Ruby on Rails). Based on the Open Food Network engine but customized for the Spanish context. | [katuma.org](https://katuma.org) |
| **Gatux** | Active | Simple and lightweight software for managing basic orders in small groups. | [github.com/gatux](https://github.com/gatux) |

## 2. Multi-Model Platforms
Platforms that allow a bridge between producers and different types of consumption groups.

| Name | Status | Technical Notes | Link |
| :--- | :--- | :--- | :--- |
| **Open Food Network (ES)** | **Active** | Global Open Source platform. In Spain, it often operates through the **Katuma** instance to support hubs and groups. | [openfoodnetwork.es](https://openfoodnetwork.es) |
| **Cagette.net** | Growing | Used by some groups in the Pyrenees and border regions with France. | [cagette.net](https://cagette.net) |

## 3. Collaborative & P2P Tools
* **Comunidades.net:** A platform used for the broader management of intentional communities and cooperatives that include consumption groups.
* **FairCoop / FairCoin:** Historically used by some radical groups for alternative payment systems and social accounting.
