# Existing Software: SoLaWi

This document maps the specialized tools currently used by the Solidarity Farming community (primarily in the DACH region) to manage budget-based agriculture.

## 1. Specialist Software
| Name | Status | Technical Notes | Link |
| :--- | :--- | :--- | :--- |
| **Juntagrico** | **Active** | Open Source (Django/Python). The benchmark for SoLaWi management. Handles member administration, subscription shares, and mandatory work-shifts. | [juntagrico.org](https://juntagrico.org/) |
| **Solawi-Manager** | Active | Open Source (Ruby/Rails). Specifically designed for the "Bieterrunde" (bidding rounds) and budget balancing. | [solawi-manager.net](https://solawi-manager.net) |
| **OpenOliva** | Active | Web-based tool for transparency and bid management in German-speaking cooperatives. | [openoliva.org](https://openoliva.org) |
| **Solawi-Tool** | Active | PHP-based simplified management for small groups and startup initiatives. | [github.com/solawitool](https://github.com/solawitool) |
| **OpenOlitor** | Active | A specialized, open-source administration platform for Solawi. It manages member shares, delivery planning, and automated invoicing. Developed primarily in Switzerland. | [openolitor.org](https://openolitor.org/) |
