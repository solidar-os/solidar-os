# Existing Software: GASAP (Belgium)

This document maps the technical tools used by the Belgian GASAP groups (Brussels and Wallonia) to manage seasonal contracts and urban collective distribution.

## 1. Specialist Software 
The Belgian network relies on a custom-built infrastructure designed to handle the specific relationship between a "Group" and a "Producer."

| Name | Status | Technical Notes | Link |
| :--- | :--- | :--- | :--- |
| **Logi-GASAP** | **Active** | Custom internal software. It manages the network's database, producer-group pairings, and contract renewals. | [reseaugasap.be](https://reseaugasap.be) |

## 2. Multi-Model Platforms
Due to the proximity to France and the global nature of some hubs, these platforms are gaining traction within the Belgian ecosystem.

| Name | Status | Technical Notes | Link |
| :--- | :--- | :--- | :--- |
| **Open Food Network (BE)** | **Active** | Global Open Source platform. Used by larger Belgian hubs and collectives that require advanced logistics and multi-producer storefronts. | [openfoodnetwork.be](https://openfoodnetwork.be) |
| **Cagette.net** | Active | Widely adopted by groups in Wallonia and near the French border for its user-friendly interface and group-order management. | [cagette.net](https://cagette.net) |

## 3. Communication & Self-Management Tools
* **Framasoft / Nextcloud:** Many GASAP groups use decentralized tools for document sharing (meeting minutes) and collaborative spreadsheets.
* **Signal / WhatsApp:** Used for the rapid coordination of the "Permanences" (shifts) at the distribution points.
