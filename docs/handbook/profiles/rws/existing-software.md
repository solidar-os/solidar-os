# Existing Software: RWS Model (Poland)

This document maps the digital tools used by the Polish *Rolnictwo Wspierane przez Społeczność* (CSA) movement to manage seasonal contracts and community-led distribution.

## 1. Specialist Software (Local & Custom)
The Polish ecosystem is transitioning from manual management to digital coordination, with some custom-built tools emerging within the social economy.

| Name | Status | Technical Notes | Link |
| :--- | :--- | :--- | :--- |
| **Koszyk RWS** | **Active** | A dedicated web-based tool developed to help Polish farmers manage member lists, shares, and pick-up points. | [rws.waw.pl](http://rws.waw.pl) |
| **Otwarty Koszyk** | Active | An open-source initiative focused on creating a transparent marketplace for local food, often used by RWS-like collectives. | [otwartykoszyk.pl](https://otwartykoszyk.pl) |

## 2. Multi-Model Platforms
Platforms used by larger Polish cooperatives or hubs that act as aggregators for multiple RWS groups.

| Name | Status | Technical Notes | Link |
| :--- | :--- | :--- | :--- |
| **Open Food Network (PL)** | **Active** | Managed by *Fundacja Wspierania Rozwoju Lokalnego*. It is the main infrastructure for groups looking for professional multi-producer management. | [openfoodnetwork.pl](https://openfoodnetwork.pl) |
| **Lokalny Rolnik** | Active | A commercial SaaS platform. While not a pure RWS (more of a Food Hub), many RWS members use it as a reference for UX and logistics. | [lokalnyrolnik.pl](https://lokalnyrolnik.pl) |

## 3. Collaborative & P2P Tools
* **Facebook Groups:** Still the primary tool for community discussion and "surplus" management within the RWS.
* **Google Sheets:** Widely used for tracking the "Bieterrunde" (bidding rounds) or seasonal payments in smaller Polish groups.
* **Portal rws.pl:** The central knowledge hub managed by the *Fundacja im. Stanisława Karłowskiego*, providing resources but not direct management software.
