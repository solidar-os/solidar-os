# Existing Software: ACP (Switzerland)

This document maps the technical tools currently used by the Federation of Regional Agriculture of Proximity (FRACP) and individual ACP cooperatives.

## 1. Specialist Software
Tools designed for the specific legal and participation requirements of the Swiss ACP model.

| Name | Status | Technical Notes | Link |
| :--- | :--- | :--- | :--- |
| **Juntagrico** | **Active** | Open Source (Django/Python). Originally for SoLaWi but widely adopted by Swiss ACPs for managing subscriptions and work-hours. | [juntagrico.org](https://juntagrico.org/) |
| **Klaus** | Active | A specialized fork of Juntagrico, customized for the logistical needs of cooperatives in the Zurich and Romandie areas. | [klaus.li](https://klaus.li) |
| **Panier.ch** | Active | A historic web platform dedicated to the Swiss French-speaking community for managing vegetable baskets. | [panier.ch](http://panier.ch) |

## 2. Multi-Model Platforms
Global or cross-border platforms used by some ACPs for their robust ordering and logistics features.

| Name | Status | Technical Notes | Link |
| :--- | :--- | :--- | :--- |
| **Open Food Network (CH)** | Active | Global Open Source platform. Used by ACPs that require advanced multi-producer coordination or integrated e-commerce features. | [openfoodnetwork.ch](https://openfoodnetwork.ch) |
| **Cagette.net** | Active | Often used by cross-border ACPs (France-Switzerland) due to its popularity in the bordering French regions. | [cagette.net](https://cagette.net) |
