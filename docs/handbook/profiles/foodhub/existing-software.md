# Existing Software: Food Hub (ireland)

This document maps the digital infrastructures used by Food Hubs globally to coordinate multi-producer aggregation and professional distribution.

## 1. Multi-Model Platforms (Open Source)
These platforms are the global standard for Food Hubs. They are designed from the ground up to handle multiple producers, inventory aggregation, and complex fee structures.

| Name | Area | Technical Notes | Link |
| :--- | :--- | :--- | :--- |
| **Open Food Network** | **Global / IE** | The benchmark for Irish hubs (e.g., *Stoneybatter Pride of Place*). Specifically designed for complex multi-vendor cycles. | [openfoodnetwork.ie](https://openfoodnetwork.ie) |
| **Cagette.net** | **FR / BE** | Widely used for its "Producteur-Référent" logic which simplifies hub coordination. | [cagette.net](https://cagette.net) |

## 2. Professional & SaaS Solutions (Commercial)
Commonly used by large-scale hubs that require advanced logistics or "Market Host" features.

| Name | Area | Technical Notes | Link |
| :--- | :--- | :--- | :--- |
| **NeighborFood** | **IE / UK** | Born in Ireland. Excellent UX for "Click & Collect" logistics. Focuses on the "Market Host" role. | [neighborfood.ie](https://neighborfood.ie) |
| **Local Food Marketplace** | **USA / CA** | Comprehensive SaaS for Food Hubs. Includes B2B features for schools and hospitals. | [localfoodmarketplace.com](https://localfoodmarketplace.com) |
| **Local Line** | **USA / CA** | Leading platform for farmers' markets and regional food hubs. | [localline.ca](https://localline.ca) |

## 3. Specialized Logistics & Resource Tools
Software used to manage the physical assets and delivery routes of the Hub, often integrated with the storefront.

* **Route4Me / OptimoRoute:** Integrated via API for home delivery fleets.
* **Skedda:** Used in physical hubs to manage the **booking of shared commercial kitchens** and production units.
* **Odoo (Inventory Module):** Used by very large hubs to manage real-time barcode scanning and warehouse placement.
